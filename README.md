# Dice's Blog
## 概要
NotionをCMSとしたメモブログ.

現在はポートフォリオとしても使用している.

## 仕様
- notionに保存ボタンを配置し押すと変更が適応される
- notionのデータベースを取得してDjangoで記事を生成している
- データは保存時にjsonファイルに保存される（今後はデータベースに保存できるようにしたい！）
- レスポンシブに対応
-

  - Djangp
